# 🌆 Web Slavonski Brod

Jednostavna web stranica o gradu  
izrađena koristeći **HTML5** i **CSS3**.

---

## 📖 O projektu

Cilj projekta je na moderan i pregledan način predstaviti grad, njegove znamenitosti, kulturu i zanimljivosti korisnicima koje zanima više informacija o gradu.

Projekt je izrađen kao studentski/web projekt s fokusom na:
- dizajn web stranice
- organizaciju sadržaja
- responzivni prikaz
- rad s HTML i CSS tehnologijama

---

## 🛠️ Korištene tehnologije

- HTML5
- CSS3
- Visual Studio Code

---

## ✨ Funkcionalnosti

✔️ Pregled osnovnih informacija o gradu  
✔️ Galerija i slike znamenitosti  
✔️ Moderan dizajn stranice  
✔️ Responzivan prikaz za različite uređaje  
✔️ Navigacija između sekcija stranice  

---

## 🖼️ Slike

Većina slika korištenih u projektu preuzeta je s interneta, dok su neke slike moje vlastite.

---

## 🎯 Namjena projekta

Projekt je namijenjen svima koji žele:
- saznati više o gradu
- istražiti znamenitosti
- pogledati vizualni prikaz grada
- upoznati kulturu i zanimljivosti grada

---

## 🚀 Pokretanje projekta

1. Preuzeti projekt
2. Otvoriti `index.html`
3. Pokrenuti u pregledniku
