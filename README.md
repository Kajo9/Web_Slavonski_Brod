# Web Slavonski Brod
## Web Slavonski Brod

Projekt izrađen kao web stranica o gradu Slavonski Brod koristeći HTML i CSS.
Cilj projekta je na jednostavan i pregledan način predstaviti grad, njegove znamenitosti, kulturu i zanimljivosti korisnicima koje zanima više informacija o gradu.

## Tehnologije
HTML5
CSS3
Sadržaj stranice

## Web stranica sadrži:

osnovne informacije o gradu
slike grada i znamenitosti
opis kulturnih i turističkih lokacija
responzivan i moderan dizajn
Slike

Većina slika korištenih u projektu preuzeta je s interneta, dok su neke slike moje vlastite.

## Namjena projekta

Projekt je namijenjen svima koji žele saznati više o gradu Slavonski Brod ili ih zanimaju njegove znamenitosti, kultura i povijest.
